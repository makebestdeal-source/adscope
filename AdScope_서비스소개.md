# AdScope - 광고 인텔리전스 플랫폼

## 서비스 개요

AdScope는 **국내 디지털 광고 시장의 모든 광고 활동을 수집, 분석, 리포팅하는 통합 광고 인텔리전스 플랫폼**입니다.

네이버, 구글, 유튜브, 메타(인스타그램/페이스북), 카카오, 틱톡 등 주요 디지털 광고 매체에서 실시간으로 광고 소재를 수집하고, AI 기반 분석을 통해 광고주별 광고 전략, 예산 추정, 타겟 분석, 캠페인 효과 측정까지 제공합니다.

---

## 핵심 기능

### 1. 광고 소재 갤러리
- **6대 채널 통합 수집**: 네이버(검색/DA/쇼핑), 구글(검색/GDN), 유튜브, 메타(FB/IG), 카카오, 틱톡
- **19,000+ 광고 소재** 축적, 일간 자동 수집
- 광고주별, 채널별, 기간별 필터링 및 검색
- 소재 이미지 + 카피 + 랜딩 URL + 광고 상품 분류

### 2. 소셜 소재 (접촉 소재)
- **AI 페르소나 기반 실제 광고 노출 시뮬레이션**
- 12개 페르소나(10~60대 남녀) × 멀티채널 서핑
- 실제 사용자가 보는 광고를 그대로 캡처
- 뉴스/기사/유튜브/쇼핑 등 자연스러운 브라우징 환경

### 3. 광고주 리포트
- **1,500+ 광고주** 프로필 관리
- 광고주별 채널 분포, 광고 소재 히스토리, 광고비 추정
- 산업 분류(20개 카테고리), 기업 프로필, 공식 채널 연동
- DART 광고선전비 연동, 메타시그널 통합 분석

### 4. 광고비 역추산
- **독자적 역추산 알고리즘**: CPC/CPM/CPV 기반 + 메타시그널 보정
- 채널별 광고비 추정: 네이버 SA, GFA, 구글 GDN, 유튜브, 메타, 카카오, 틱톡
- 월간 총 추정 광고비 **17.3억원** 규모 트래킹
- 실제 집행 데이터(160건) 기반 보정 계수 적용

### 5. 캠페인 분석
- **자동 캠페인 빌드**: 광고 소재 → 캠페인 자동 그룹핑
- 크로스채널 병합: 같은 광고주 + 같은 상품 = 하나의 캠페인
- 캠페인 저니(Journey): 노출 → 관심 → 고려 → 전환 퍼널
- 캠페인 리프트(Lift): Pre/Post 7일 검색/소셜/매출 변화 측정

### 6. 시장 분석
- **산업별 현황**: 20개 산업 카테고리별 광고 트렌드
- **경쟁사 비교**: 동종 업계 내 광고비/채널/소재 비교
- **제품/서비스별 분석**: SKU 레벨 광고 추적
- **광고주 트렌드**: 신규/증가/감소 광고주 모니터링

### 7. 소셜 인사이트
- **브랜드 버즈**: 뉴스/소셜 멘션 모니터링 + 감성 분석
- **소비자 인사이트**: 광고 카피 분석, 프로모션 트렌드, 위닝 크리에이티브
- **타겟 오디언스**: 페르소나별 광고 노출 분석, 채널 최적화 추천
- **캠페인 효과**: Before/After 비교, 리프트 측정, 다중 캠페인 비교

### 8. 매체/광고주 마스터 인덱스
- **120+ 디지털 광고 매체** 데이터베이스 (네이버/구글/메타부터 DSP/SSP/리워드까지)
- **1,500+ 광고주** 마스터 DB (대기업~중소형, 외국 브랜드 포함)
- 매체별 과금 체계, MAU, 운영사 정보
- 광고주별 공식 웹사이트, 소셜 채널 URL, 산업 분류

## 차별점

1. **올인원 플랫폼**: 소재 수집 + 광고비 추정 + 캠페인 분석 + 소셜 인사이트를 하나의 플랫폼에서
2. **AI 페르소나**: 실제 사용자처럼 광고를 경험하는 12개 AI 페르소나
3. **크로스채널 분석**: 6대 매체를 통합 관리, 크로스채널 캠페인 자동 병합
4. **실시간 모니터링**: 매일 자동 수집, 신규 광고/광고주 즉시 감지
5. **한국 시장 특화**: 네이버/카카오 등 국내 매체 심층 지원, 한국어 UI
